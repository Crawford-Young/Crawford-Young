# CLAUDE.md — Crawford-Young profile repo

GitHub profile README + self-owned stats generator. Every image in the README is a file in `assets/`, generated by code in this repo — zero third-party image services. Workspace standards (`~/code/CLAUDE.md`) apply, **lean scope override**: no Husky, commitlint, coverage gate, Justfile, or Dependabot here.

## Stack

TypeScript ESM (NodeNext, strict, `.js` import extensions) · tsx runner · pnpm · Vitest (snapshot + unit, no coverage gate).

## Commands

| Command | What |
|---|---|
| `pnpm test` | 29 tests: renderer snapshots + streak/aggregation units |
| `pnpm generate` | Fetch GitHub GraphQL → write all `assets/*.svg` (needs `STATS_TOKEN` env — fine-grained read-only PAT, never commit) |
| `pnpm export-claude` | Parse `~/.claude/projects/**/*.jsonl` → `data/claude-usage.json` (aggregate daily token counts only — no conversation content) |
| `pnpm exec tsc --noEmit` | Typecheck |

## Architecture

- `src/theme.ts` — tokyonight palette + card frame helpers
- `src/github.ts` — GraphQL fetchers; contribution calendar walked year-by-year (boundary days sum-merged, 1ms window advance)
- `src/cards.ts` — stats / languages / streak / activity / claude renderers
- `src/badges.ts` — flat-square badges, simple-icons paths vendored at build time
- `src/claude.ts` — JSONL dedupe by message id, bucket by local date, invalid timestamps skipped BEFORE marking id seen
- `src/generate.ts` — Action entry; claude.svg renders only if `data/claude-usage.json` exists (machine off = stale card, never broken)
- `src/export-claude.ts` + `scripts/export-claude.ps1` — local export, run by Windows Scheduled Task `ClaudeUsageExport` daily 6PM (pull --rebase → export → commit+push if staged diff, one rebase retry)

## Automation

`.github/workflows/update-stats.yml` — daily 06:00 UTC + manual dispatch; test → generate → commit `assets/` diff as `chore: update stats [skip ci]`. Secret `STATS_TOKEN` set in repo settings.

## Rules

- README references `./assets/` paths only — never reintroduce external image URLs
- Renderer changes: update snapshots deliberately (`vitest run -u`), verify SVG textually (grep `NaN|Infinity|undefined`, sane width/height) before committing
- `data/claude-usage.json` must contain only `YYYY-MM-DD` keys → `{input, output}` integers — no strings, ever
